cmaq2hemco User's Guide
========================

Utilities for converting CMAQ-ready emissions to HEMCO supported files.


Getting Started
---------------

The best way to get started is to install (see below) and then explore the
examples gallery.


Installation
------------

`cmaq2hemco` is not yet available through pypi.org because it is still in
rapid development. You can get the latest version with this command.

.. code-block::

    pip install git+https://github.com/barronh/cmaq2hemco.git


Issues
------

If you're having any problems, open an issue on github.

https://github.com/barronh/cmaq2hemco/issues
